# Product Templates — Starter Kit

Each file is a template. Copy this folder to a new project and fill in.

## Files

| File | Purpose |
|------|---------|
| `vision.md` | Core product vision statement |
| `strategy.md` | Go-to-market and growth strategy |
| `objectives.md` | Business objectives with measurable KPIs |
| `scope.md` | In-scope and out-of-scope features |
| `personas.md` | User personas with goals and pain points |
| `stakeholders.md` | Stakeholder map with influence/interest |
| `constraints.md` | Business, technical, and operational constraints |
| `assumptions.md` | Key assumptions driving decisions |
| `risks.md` | Risk register with mitigations |
| `success-criteria.md` | Tangible success criteria per phase |
