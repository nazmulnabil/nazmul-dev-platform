# Scope

## In Scope

- Email/password registration and login
- JWT access + refresh token management
- User profile view and update
- Address CRUD with default address management
- Product categories (hierarchical tree)
- Products with images and variants
- Multi-warehouse inventory with reservation lifecycle
- Spec-first development workflow

## Out of Scope (Phase 1)

- Social login / OAuth
- Shopping cart
- Checkout and payments
- Order management
- Seller/vendor dashboards
- Admin dashboard
- Search (beyond basic filtering)
- Mobile apps
- Internationalization

## Phase Boundaries

| Phase | In Scope | Out of Scope |
|-------|----------|--------------|
| Phase 1 | Auth, users, catalog | Cart, checkout, orders, payments |
| Phase 2 | Cart, checkout, orders | Multi-vendor, marketplace |
| Phase 3 | Multi-vendor, marketplace | Admin analytics, advanced search |
