# Technology Stack

| Layer | Technology | Justification |
|-------|------------|---------------|
| Backend | Django 5 + DRF | Rapid development, mature ORM |
| Database | PostgreSQL | Reliability, JSON support, full-text search |
| Auth | JWT (simplejwt) | Stateless, mobile-friendly |
| Frontend | React + Vite | Modern tooling, fast builds |
| Testing | pytest + model-bakery | Django best practices |
| API Docs | drf-spectacular | OpenAPI 3.0 schema generation |
