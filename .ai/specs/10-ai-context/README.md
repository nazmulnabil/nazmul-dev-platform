# AI Context — Files for AI Agents

These files provide AI agents with the system, product, and coding context they need without cluttering spec READMEs.

## Files

| File | Purpose |
|------|---------|
| `system-context.md` | High-level system overview, repo structure, entrypoints |
| `product-context.md` | Product goals, users, business value |
| `architecture-context.md` | Architecture patterns, key decisions, constraints |
| `coding-context.md` | Coding conventions, naming, tools, linting |
| `sprint-context.md` | Current sprint goals, active stories, blockers |
| `known-issues.md` | Known bugs, gotchas, workarounds |
| `implementation-context.md` | Implementation notes, partial work, decisions made mid-sprint |
